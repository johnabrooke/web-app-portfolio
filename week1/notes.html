Five localstorage methods

  • setItem() Add key and value to localStorage
  • getItem() Retrieve a value by it's key from localStorage
  • removeItem() Remove and item by it's key from localStorage
  • clear() Clear everything stored in localStorage
  • key() Pass a number to retrieve nth key of an item in localStorage (ex. loop through keys to find a name)
